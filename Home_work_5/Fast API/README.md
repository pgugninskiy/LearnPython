# Домашнее задание: Docker контейнер с веб-приложением на FastAPI

## 📋 Описание проекта
Это учебный проект, выполненный в рамках курса по Python. 
Цель работы: создание изолированного Docker-контейнера с веб-сервером на базе фреймворка **FastAPI**.

Приложение реализует простой эндпоинт для проверки работоспособности сервиса (Health Check).

### Основные возможности
- Запуск веб-сервера **Uvicorn** внутри контейнера.
- Эндпоинт `/ping/`, возвращающий JSON-ответ `{"message": "pong"}`.
- Полная изоляция зависимостей через виртуальное окружение внутри образа.
- Совместимость с инструментами **Podman**.

---

## 🛠 Технологический стек
- **Язык:** Python 3.11
- **Фреймворк:** FastAPI
- **Сервер:** Uvicorn
- **Контейнеризация:** Docker / Podman
- **Управление зависимостями:** pip + requirements.txt

---

## 🚀 Быстрый старт

### Требования
Для запуска проекта необходимо наличие установленного движка контейнеризации:
- [Docker Desktop](https://www.docker.com/products/docker-desktop/)
- **ИЛИ** [Podman](https://podman.io/) (рекомендуется для Linux/macOS, также работает на Windows)

### 1. Клонирование репозитория
```bash
git clone <URL_ВАШЕГО_РЕПОЗИТОРИЯ>
cd <ПАПКА_ПРОЕКТА>
```

### 2. Сборка образа
Вы можете использовать любую удобную команду. Образ будет собран одинаково в обоих случаях.

**Вариант для Docker:**
```bash
docker build -t homework-fastapi .
```

**Вариант для Podman:**
```bash
podman build -t homework-fastapi .
```

### 3. Запуск контейнера
Запускаем контейнер с пробросом порта `8001` на локальную машину.

**Docker:**
```bash
docker run -p 8001:8001 homework-fastapi
```

**Podman:**
```bash
podman run -p 8001:8001 homework-fastapi
```

> **Примечание:** Сервер будет доступен по адресу `http://localhost:8001`.

---

## 🧪 Проверка работы

После запуска контейнера проверьте работоспособность API:

1. Откройте браузер или используйте `curl`.
2. Перейдите по ссылке: **[http://localhost:8001/ping/](http://localhost:8001/ping/)**

**Ожидаемый ответ:**
```json
{
  "message": "pong"
}
```

Также доступна автоматическая документация Swagger UI по адресу:  
[http://localhost:8001/docs](http://localhost:8000/docs)


## 📝 Примечания для разработчика
- Виртуальное окружение (`venv`) **не включено** в репозиторий согласно лучшим практикам.
- Порт внутри контейнера настроен на `8001`, директива `EXPOSE` добавлена в `Dockerfile`.
- Для установки зависимостей используется зеркало `pypi.tuna.tsinghua.edu.cn` (опционально) для ускорения сборки в условиях нестабильного сети, либо стандартный PyPI с увеличенным таймаутом.
