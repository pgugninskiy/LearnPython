## 📐 1. Формула системно-функционального определения

Вы верно записали структуру. Давайте оформим её как **шаблон для применения**:

```
ОБЪЕКТ = [Надсистемный признак] + [Главная функция] + [Рабочий орган] + [Подсистемы]
```

| Шаг | Вопрос | Пример (Лампа) |
|-----|--------|----------------|
| **1** | **Назначение (ГФ)** | Освещение предметов |
| **2** | **Надсистемный признак** | Осветительные приборы |
| **3** | **Рабочий орган** | Световое пятно (излучатель) |
| **4** | **Подсистемы** | Стойка, лампочка, провод, патрон, выключатель |

> 💡 **Важно:** Рабочий орган — это то, что **непосредственно воздействует на изделие**. Для лампы это не «лампочка», а **свет** (или вольфрамовая нить/LED как источник света).

## 📝 2. Исправленные примеры (по шаблону)

Вот как можно оформить ваши примеры в едином формате:

### 🪑 Настольная лампа
| Компонент | Формулировка |
|-----------|--------------|
| **Надсистемный признак** | Осветительный прибор |
| **Главная функция** | **Освещает** предметы (изменяет видимость) |
| **Рабочий орган** | **Источник света** (нить/LED) + **Отражатель** |
| **Подсистемы** | Стойка, патрон, провод, выключатель, корпус |

> **Полное определение:** *Настольная лампа — это осветительный прибор для освещения предметов с помощью светового излучения, состоящий из источника света, патрона, стойки, провода и выключателя.*

---

### 🚗 Автомобиль
| Компонент | Формулировка |
|-----------|--------------|
| **Надсистемный признак** | Транспортное средство |
| **Главная функция** | **Перемещает** пассажиров/грузы (изменяет положение в пространстве) |
| **Рабочий орган** | **Колёса** (контакт с дорогой) |
| **Подсистемы** | Двигатель, трансмиссия, салон, тормоза, выхлопная система, рулевое управление |

> **Полное определение:** *Автомобиль — это транспортное средство для перемещения пассажиров и грузов с помощью колёс, состоящее из двигателя, трансмиссии, ходовой части, салона и систем управления.*

---

### 🔪 Нож
| Компонент | Формулировка |
|-----------|--------------|
| **Надсистемный признак** | Кухонный инструмент / Режущий прибор |
| **Главная функция** | **Разделяет** пищу (изменяет целостность объекта) |
| **Рабочий орган** | **Лезвие** (режущая кромка) |
| **Подсистемы** | Рукоятка, клинок, заклёпки/крепления |

> **Полное определение:** *Нож — это кухонный инструмент для разделения пищи с помощью режущей кромки лезвия, состоящий из клинка и рукоятки.*

## 🕰 3. Системно-диалектический подход (Горизонталь развития)

Вы отметили: *«Система (вчера) → Система (сегодня) → Система (завтра)»*

Это **эволюция систем** по законам ТРИЗ. Давайте оформим как таблицу:

| Этап | Характеристика | Пример (Лампа) | Пример (Автомобиль) |
|------|----------------|----------------|---------------------|
| **Вчера** | Исходная система, есть недостатки | Свеча / Керосиновая лампа | Лошадь + Телега |
| **Сегодня** | Текущая система, решает задачу | Электрическая лампа накаливания | Двигатель внутреннего сгорания |
| **Завтра** | Новая система, без недостатков | LED / Умное освещение | Электромобиль / Беспилотник |

---

## 🔄 4. Функциональная «жизнь» Систем

Вы записали цепочку развития:
```
Потребность → Функция → Исходная система → Функциональный недостаток → Переход к новой системе
```

Это **алгоритм эволюции**. Давайте развернём на примере:

### Пример: Эволюция освещения

| Этап | Что происходит | Пример |
|------|----------------|--------|
| **1. Потребность** | Человек хочет видеть в темноте | «Нужно освещение ночью» |
| **2. Функция** | Формулируется главная функция | «Освещать пространство» |
| **3. Исходная система** | Появляется первое решение | Костёр / Факел |
| **4. Функциональный недостаток** | Обнаруживаются вредные функции | Дымит, опасно, неудобно |
| **5. Переход к новой системе** | Эволюция через разрешение противоречий | Свеча → Лампа накаливания → LED |

## ✅ Чек-лист для системно-функционального определения

Когда определяете объект, проверьте:

| Вопрос | Если «НЕТ» → исправьте |
|--------|------------------------|
| **Назван ли надсистемный признак?** | К какому классу относится объект? |
| **Сформулирована ли ГФ через глагол?** | Не «для света», а «освещает предметы» |
| **Указан ли рабочий орган?** | Что напрямую воздействует на изделие? |
| **Перечислены ли ключевые подсистемы?** | Без лишних деталей, только значимые |
| **Видна ли эволюция (вчера/сегодня/завтра)?** | Как система развивалась и куда движется? |

## 💡 Главный инсайт урока

> **Определение объекта — это не описание того, что он есть.**
> **Это описание того, что он делает и как развивается.**

Когда вы правильно определяете систему — вы уже видите, **куда она будет эволюционировать** и **какие недостатки устранять**.
