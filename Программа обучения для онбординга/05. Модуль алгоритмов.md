# Программа обучения: "Работа с алгоритмами"

## **Модуль 1: Создание конфигурации алгоритма**
- **Цель:** Научить создавать конфигурацию алгоритма.
- **Темы:**
  ### Шаг 1: Переход к созданию конфигурации
  - Нажатие на кнопку "Создать конфигурацию".
  - Открытие модального окна для создания новой конфигурации.

  ### Шаг 2: Заполнение обязательных полей
  - Выбор типа конфигурации "Алгоритм".
  - Ввод наименования алгоритма.
  - Ввод описания алгоритма.

  ### Шаг 3: Подтверждение создания
  - Нажатие на кнопку "Создать".
  - Система создает конфигурацию типа "Алгоритм".

  ### Интерфейс конфигуратора
  - Левая часть интерфейса: элементы алгоритма.
  - Центральная часть: рабочая область для конфигурирования.
  - Правая часть: свойства блоков.

---

## **Модуль 3: Добавление параметров и переменных**
- **Цель:** Научить добавлять параметры и переменные в алгоритм.
- **Темы:**
  ### Шаг 1: Работа с блоком "Начало"
  - Нажатие на блок "Начало" для отображения свойств.
  - Добавление параметров через кнопку "+ Добавить".

  ### Шаг 2: Настройка параметра/переменной
  - Ввод пользовательского названия параметра:
    - Длина: от 1 до 30 символов.
    - Разрешенные символы: 'A'-'Z', 'a'-'z', '0'-'9', '_'.
    - Не должно начинаться с цифры.
  - Выбор направления передачи значения:
    - IN, OUT, IN_OUT, LOCAL.
  - Выбор вида параметра:
    - ATOM, ATOM_ARRAY, OBJECT, OBJECT_ARRAY, TUPLE, TUPLE_ARRAY.

  ### Шаг 3: Проверка добавленных параметров
  - Система отображает сконфигурированные параметры и переменные в списке.

---

## **Модуль 4: Работа с элементами алгоритма**
- **Цель:** Научить добавлять и настраивать элементы алгоритма.
- **Темы:**
  ### Добавление блока
  - Перетаскивание элемента из левой части в рабочую область.
  - Удаление элемента через контекстное меню.

  ### Добавление связи
  - Нажатие на стрелку в нижней части элемента.
  - Перетаскивание стрелки к связываемому элементу. 
  - Удаление связи через контекстное меню.

---

## **Модуль 5: Настройка свойств блоков**
- **Цель:** Научить настраивать свойства различных блоков.
- **Темы:**
  ### Блок "Объект"
  - Создание нового экземпляра объекта.
  - Загрузка существующего экземпляра объекта.
  - Настройка локальной переменной и параметров.

  ### Блок "Запрос"
  - Выбор конфигурации запроса.
  - Настройка параметров запроса.
  - Сохранение результатов в локальную переменную.

  ### Блок "Пользовательское действие"
  - Настройка динамической формы или конфигурации UI.
  - Добавление полей и кнопок формы.
  - Настройка ограничений и масок для полей.

  ### Блок "Вычисление"
  - Использование редактора выражений.
  - Добавление переменных и выражений.

  ### Блок "Правило"
  - Выбор конфигурации правила.
  - Настройка параметров и условий.

  ### Блок "Условие"
  - Задание логического выражения.
  - Разработка веток "Да" и "Нет".

  ### Блок "Сохранение результата"
  - Настройка сохранения результата выполнения алгоритма.

---

## **Модуль 6: Генерация ссылки для BIRT**
- **Цель:** Научить генерировать ссылки для использования в BIRT.
- **Темы:**
  - Нажатие на блок "Начало" для отображения свойств.
  - Генерация ссылки через кнопку "Сгенерировать ссылку для BIRT".
  - Копирование ссылки с параметрами или с константами.

---

## **Модуль 7: Практические задания**
- **Цель:** Закрепить знания на практике.
- **Задания:**
  1. Создать новый алгоритм.
  2. Добавить параметры и переменные.
  3. Настроить блок "Объект" для создания экземпляра.
  4. Настроить блок "Запрос" для получения данных.
  5. Добавить блок "Условие" и разработать ветки "Да" и "Нет".
  6. Сохранить результат выполнения алгоритма.
