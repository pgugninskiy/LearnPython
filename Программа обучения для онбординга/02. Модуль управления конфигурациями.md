Модуль управления конфигурациями (МУК)
Создание новой конфигурации.
Импорт/экспорт конфигураций.
Управление версиями.

МУК используется для работы со следующими типами конфигураций:

Конфигурации Атрибутов и Объектов. Обеспечивает реализацию визуального редактора для управления конфигурациями атрибутов и объектов.
Конфигурации Запросов. Обеспечивает реализацию визуального редактора для управления конфигурациями запросов.
Модуль Алгоритмов. Обеспечивает реализацию визуального редактора для управления конфигурациями алгоритмов.
Модуль Правил. Обеспечивает реализацию визуального редактора для управления конфигурациями правил.
Модуль UI. Обеспечивает реализацию визуального редактора для управления конфигурациями экранов, меню, стартовой страницы.
Модуль Отчетов. Обеспечивает реализацию визуального редактора для управления конфигурациями отчетов.
Модуль Расписания и Задач. Обеспечивает реализацию визуального редактора для управления конфигурациями задач.
Модуль Ролей и Привилегий. Обеспечивает реализацию визуального редактора для управления конфигурациями ролями, привилегиями, функциями.


# Программа обучения : "Работа с модулем управления конфигураций"

## Модуль 1: Введение в систему
- **Цель:** Познакомить сотрудников с основными понятиями и интерфейсом системы.
- **Темы:**
  - Что такое дерево конфигураций.
  - Основные элементы интерфейса: группы, подгруппы, конфигурации.
  - Значение уникальности наименований для конфигураций.
  - Обзор типов конфигураций и их назначения.

---

## Модуль 2: Работа с группами
- **Цель:** Научить создавать, редактировать и управлять группами конфигураций.
- **Темы:**
  ### Создание корневой группы:
  - Переход в корень иерархии.
  - Создание новой группы через модальное окно.
  - Заполнение поля "Наименование".
  
  ### Создание подгруппы:
  - Переход в корневую группу.
  - Процесс создания вложенной подгруппы.
  
  ### Редактирование наименования группы:
  - Как изменить название группы.
  - Использование модального окна для редактирования.
  
  ### Перемещение группы:
  - Выбор целевой папки.
  - Подтверждение перемещения.

---

## Модуль 3: Поиск и фильтрация
- **Цель:** Научить эффективно находить нужные группы и конфигурации.
- **Темы:**
  ### Поиск по группам/конфигурациям:
  - Использование поля "Поиск".
  - Выбор метода поиска из выпадающего списка.
  - Анализ результатов поиска (превью и полные результаты).
  
  ### Фильтрация конфигураций:
  - Фильтрация по типу, версии и статусу.
  - Применение критериев фильтрации.
  - Сброс фильтра.

---

## Модуль 4: Управление конфигурациями
- **Цель:** Научить создавать, редактировать и управлять конфигурациями.
- **Темы:**
  ### Создание конфигурации:
  - Переход в выбранную группу.
  - Заполнение обязательных полей (тип, наименование, описание).
  - Уникальность наименования.
  
  ### Просмотр списка конфигураций:
  - Отображение информации о конфигурациях (тип, наименование, статус и т.д.).
  
  ### Редактирование наименования конфигурации:
  - Изменение имени в версии "Черновик".
  - Уникальность нового имени.
  
  ### Перемещение конфигурации:
  - Выбор целевой группы.
  - Подтверждение перемещения.
  
  ### Копирование конфигурации:
  - Копирование без связанных конфигураций.
  - Ввод нового имени и выбор группы для размещения.

---

## Модуль 5: Работа с версиями
- **Цель:** Научить работать с черновиками и рабочими версиями конфигураций.
- **Темы:**
  - Редактирование черновика.
  - Просмотр в режиме "Только чтение".
  - Просмотр идентификатора конфигурации.
  - Просмотр связей конфигурации.

---

## Практические задания
- **Цель:** Закрепить знания на практике.
- **Задания:**
  - Создать корневую группу и подгруппу.
  - Найти конкретную конфигурацию с помощью поиска.
  - Создать новую конфигурацию и отредактировать ее имя.
  - Переместить конфигурацию в другую группу.
  - Скопировать конфигурацию и проверить уникальность имени.
  - Просмотреть связи выбранной конфигурации.
