# Программа обучения: "Работа с правилами"

## **Модуль 1: Создание конфигурации правила**
- **Цель:** Научить создавать конфигурацию правила.
- **Темы:**
  ### Шаг 1: Переход к созданию конфигурации
  - Нажатие на кнопку "Создать конфигурацию".
  - Открытие модального окна для создания новой конфигурации.

  ### Шаг 2: Заполнение обязательных полей
  - Выбор типа конфигурации "Правило".
  - Ввод наименования конфигурации.
  - Ввод описания конфигурации.

  ### Шаг 3: Подтверждение создания
  - Нажатие на кнопку "Создать".
  - Система создает конфигурацию типа "Правило".

---

## **Модуль 3: Добавление переменных**
- **Цель:** Научить добавлять переменные для использования в правилах.
- **Темы:**
  ### Шаг 1: Добавление переменной
  - Нажатие на кнопку "+ Добавить переменную".
  - Заполнение свойств переменной:
    - Код переменной:
      - Длина: до 30 символов.
      - Разрешенные символы: латинские буквы, цифры, "_".
      - Не должно начинаться с цифры или символа подчеркивания.
    - Тип данных:
      - Строка.
      - Целое число.
      - Дробное число.
      - Логическое значение.
      - Объект.
    - Описание переменной (необязательное поле).
    - Значение по умолчанию (необязательное поле).

  ### Шаг 2: Сохранение переменной
  - Нажатие на кнопку "Сохранить".
  - Система отображает добавленную переменную в списке.

---

## **Модуль 4: Добавление строк и колонок**
- **Цель:** Научить добавлять строки и колонки для формирования условий.
- **Темы:**
  ### Добавление строки
  - Нажатие на кнопку "Добавить строку".
  - Задание названия строки (по умолчанию: "Строка + позиция").
  - Установка позиции строки.
  - Активация/деактивация строки.
  - Сохранение строки.

  ### Добавление колонки
  - Нажатие на кнопку "Добавить колонку".
  - Задание условия в колонке.
  - Использование выражений и операторов для формирования условий.

---

## **Модуль 5: Настройка условий и результатов**
- **Цель:** Научить настраивать условия и результаты правил.
- **Темы:**
  ### Настройка условий
  - Формирование условий с использованием переменных.
  - Применение логических операторов (AND, OR, NOT).
  - Сравнение значений полей объекта.

  ### Настройка результатов
  - Задание действий, которые будут выполнены при выполнении условий.
  - Примеры действий:
    - Автоматический расчет формул.
    - Управление правами пользователей.
    - Отображение доступности кнопок.
    - Рассылка сообщений.

---

## **Модуль 6: Тестирование правил**
- **Цель:** Научить тестировать правила перед их использованием.
- **Темы:**
  - Проверка корректности условий.
  - Проверка выполнения действий.
  - Анализ результатов тестирования.

---

## **Модуль 7: Практические задания**
- **Цель:** Закрепить знания на практике.
- **Задания:**
  1. Создать новое правило.
  2. Добавить переменные для использования в правиле.
  3. Настроить строки и колонки для формирования условий.
  4. Задать условия и результаты для правила.
  5. Протестировать правило и проанализировать результаты.
