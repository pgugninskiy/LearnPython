# Программа обучения: "Работа с UI"

## **Модуль 1: Создание конфигурации UI**
- **Цель:** Научить создавать конфигурацию UI.
- **Темы:**
  ### Шаг 1: Переход к созданию конфигурации
  - Нажатие на кнопку "Создать конфигурацию".
  - Открытие модального окна для создания новой конфигурации.

  ### Шаг 2: Заполнение обязательных полей
  - Выбор типа конфигурации "Форма".
  - Ввод наименования конфигурации.
  - Ввод описания конфигурации.

  ### Шаг 3: Подтверждение создания
  - Нажатие на кнопку "Создать".
  - Система создает конфигурацию типа "Форма".

---

## **Модуль 3: Работа с формами UI**
- **Цель:** Научить создавать и настраивать основные формы, модальные окна и фрагменты UI.
- **Темы:**
  ### Основная форма
  - Выбор вида конфигурируемой формы: "Основная форма".
  - Основная форма всегда видна пользователю и служит для конфигурирования рабочих экранов.

  ### Модальное окно
  - Выбор вида конфигурируемой формы: "Модальное окно".
  - Настройка заголовка модального окна:
    - Задание заголовка.
    - Установка заголовка от переменной.
    - Выбор переменной для формирования заголовка.

  ### Фрагмент UI
  - Выбор вида конфигурируемой формы: "Фрагмент UI для алгоритмов".
  - Особенности: вызывается через алгоритм, а не напрямую через форму UI.

---

## **Модуль 4: Разметка и компоновка элементов**
- **Цель:** Научить создавать разметку и размещать элементы в рабочей области.
- **Темы:**
  ### Добавление разметки
  - Перетаскивание блочной разметки из левой части экрана в рабочую область.
  - Настройка выравнивания (по вертикали, горизонтали или комбинированное).

  ### Вложенные элементы
  - Добавление разметки внутри ранее добавленной.
  - Настройка отступов для элементов.

---

## **Модуль 5: Работа с переменными**
- **Цель:** Научить создавать и настраивать переменные для работы в UI.
- **Темы:**
  ### Добавление переменной
  - Нажатие на кнопку "+ Добавить переменную".
  - Заполнение свойств переменной:
    - Название.
    - Тип данных.
    - Значение по умолчанию.
    - Маска ввода (для строк/чисел).
    - Мин/макс значение (для чисел).

  ### Особые параметры
  - Обязательная – параметр для валидации.
  - Константа – значение переменной не изменяется.
  - Сохранение в памяти – значение сохраняется в SessionStorage.

---

## **Модуль 6: Добавление компонентов**
- **Цель:** Научить добавлять и настраивать компоненты в UI.
- **Темы:**
  ### Кнопка
  - Добавление кнопки в рабочую область.
  - Настройка свойств кнопки:
    - Заголовок.
    - Действия по нажатию (алгоритм, форма, модальное окно и т.д.).
    - Иконка слева/справа.
    - Размер и отступы.

  ### Поле ввода
  - Добавление поля ввода.
  - Настройка свойств:
    - Текст внутри.
    - Вариант оформления.
    - Активность (да/нет).
    - Переменные для записи и чтения.

  ### InlineMessage
  - Добавление компонента "InlineMessage".
  - Настройка текста сообщения, статуса (Success, Error, Warning) и отступов.

  ### Typography
  - Добавление компонента "Typography".
  - Настройка текста, размера шрифта, толщины и отступов.

---

## **Модуль 7: Настройка источников данных**
- **Цель:** Научить подключать и настраивать источники данных.
- **Темы:**
  ### Добавление источника данных
  - Нажатие на кнопку "+ Добавить источник данных".
  - Выбор запроса из доступных источников данных.
  - Настройка параметров запроса (связь с переменными UI).

  ### Автоматическое обновление
  - Указание интервала автоматического обновления (в секундах).

---

## **Модуль 8: Практические задания**
- **Цель:** Закрепить знания на практике.
- **Задания:**
  1. Создать новую конфигурацию UI.
  2. Добавить основную форму и настроить заголовок.
  3. Добавить модальное окно и настроить заголовок от переменной.
  4. Создать переменные для хранения данных пользователя.
  5. Добавить кнопку и настроить действия по нажатию.
  6. Добавить поле ввода и настроить маску ввода.
  7. Настроить источник данных для таблицы.
