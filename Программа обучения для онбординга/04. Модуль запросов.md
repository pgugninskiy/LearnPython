# Программа обучения: "Работа с запросами"

## **Модуль 1: Введение в работу с запросами**
- **Цель:** Познакомить сотрудников с основными понятиями и назначением запросов.
- **Темы:**
  - Что такое запросы и их роль в системе.
  - Предусловия для создания запросов:
    - Конфигурация запроса.
    - Наименование и описание конфигурации.
    - Список объектов и атрибутов.
    - Параметры запроса (наименование, тип, значение по умолчанию).
    - Условия компоновки параметров.
    - Ожидаемый результат запроса.

---

## **Модуль 2: Создание конфигурации запроса**
- **Цель:** Научить создавать конфигурацию запроса.
- **Темы:**
  ### Шаг 1: Переход к созданию конфигурации
  - Нажатие на кнопку "Создать конфигурацию".
  - Открытие модального окна для создания новой конфигурации.

  ### Шаг 2: Заполнение обязательных полей
  - Выбор типа конфигурации "Запрос".
  - Ввод наименования конфигурации.
  - Ввод описания конфигурации.

  ### Шаг 3: Подтверждение создания
  - Нажатие на кнопку "Создать".
  - Система создает конфигурацию типа "Запрос".

---

## **Модуль 3: Добавление объектов и атрибутов**
- **Цель:** Научить добавлять объекты и атрибуты в запрос.
- **Темы:**
  ### Шаг 1: Добавление объектов
  - Нажатие на кнопку "Новый запрос".
  - Нажатие на кнопку "Добавить объект".
  - Выбор объекта из выпадающего списка.
  - Если объекта нет в списке, создать его отдельно.

  ### Шаг 2: Добавление атрибутов
  - Нажатие на кнопку "Добавить атрибуты".
  - Выбор одного или нескольких атрибутов из выпадающего списка.
  - Если атрибута нет в списке, создать его отдельно и добавить в объект.

  ### Шаг 3: Проверка выбранных атрибутов
  - Система отображает выбранные атрибуты и их тип данных.
  - Формат наименования атрибутов: `"Объект" + порядковый номер объекта + "Атрибут" + порядковый номер атрибута`.

  ### Шаг 4: Работа со связанными объектами
  - Если выбран атрибут с типом ID, система автоматически добавляет связанный объект.
  - Таблица "Связанные объекты" отображает:
    - Первый выбранный объект.
    - Связанный объект.
    - Связь через атрибут.

---

## **Модуль 4: Добавление параметров**
- **Цель:** Научить добавлять параметры для наложения ограничений на запрос.
- **Темы:**
  ### Шаг 1: Добавление параметров
  - Нажатие на кнопку "Параметры".
  - Нажатие на кнопку "Добавить параметр".
  - Заполнение названия параметра:
    - Должно начинаться с латинской буквы.
    - Разрешенные символы: латинские буквы, цифры, "_".
  - Выбор типа параметра из выпадающего списка.
  - Заполнение значения по умолчанию (необязательное поле).

  ### Шаг 2: Сохранение параметров
  - Нажатие на кнопку "Применить".
  - Система сохраняет добавленные параметры.

---

## **Модуль 5: Добавление условий**
- **Цель:** Научить добавлять условия для фильтрации данных.
- **Темы:**
  ### Шаг 1: Выбор типа соединения
  - Выбор типа соединения для связанных таблиц:
    - `LEFT JOIN` — все записи из левой таблицы.
    - `RIGHT JOIN` — все записи из правой таблицы.
    - `INNER JOIN` — только записи, удовлетворяющие условию объединения.

  ### Шаг 2: Создание условия связи
  - Нажатие на кнопку для создания условия связи.
  - Задание дополнительной связи между атрибутами объектов.
  - Нажатие на кнопку "Применить".

  ### Шаг 3: Добавление условий
  - Нажатие на кнопку "Добавить условия".
  - Ввод символа ":" для отображения добавленных параметров.
  - Использование автокомплита для выбора функций.
  - Использование операторов `AND` и `OR`.

---

## **Модуль 6: Настройка свойств запроса**
- **Цель:** Научить настраивать свойства запроса, объектов и атрибутов.
- **Темы:**
  ### Шаг 1: Настройка запроса
  - Исключение повторяющихся значений (чекбокс "Исключить повторы").
  - Ограничение выборки (чекбокс "Ограничить выборку").
  - Ввод значения для предупреждения (по умолчанию — 500).

---

## **Модуль 7: Генерация ссылки для BIRT**
- **Цель:** Научить генерировать ссылки для использования в BIRT.
- **Темы:**
  - Нажатие на кнопку "Ссылка на запрос".
  - Копирование ссылки (с параметрами или с константами).

---

## **Модуль 8: Практические задания**
- **Цель:** Закрепить знания на практике.
- **Задания:**
  1. Создать новый запрос.
  2. Добавить объекты и атрибуты в запрос.
  3. Добавить параметры для запроса.
  4. Настроить условия фильтрации.
  5. Настроить свойства запроса и задать алиасы.
  6. Сгенерировать ссылку для BIRT.
